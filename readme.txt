Para compilar o montador, assim como pedido nas instruções do trabalho, executa-se a seguinte linha de comando no terminal:

g++ .\montador.cpp

e para executar o arquivo objeto gerado(realizar a montagem):

.\a.exe [nome do arquivo .asm sem os colchetes]

---------------

Para o simulador, o processo é análogo:

g++ .\simulador.cpp

e para executar:

.\a.exe [nome do arquivo .obj sem os colchetes]
